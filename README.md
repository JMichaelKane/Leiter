## Leiter

一个基于 NEKit 的网络 Proxy App。

## Usage

执行：
```
> carthage update --no-use-binaries --platform ios
> pod install --repo-update
> open Leiter.xcworkspace
```

***此项目如果要在App上使用，需要99美元的开发者账号***

支持的协议：
- Http(s)
- Shadowsock
- Socks5
